# Node.JS Web Server

Study project.

"The Complete Node.js Developer Course (2nd Edition) " (Udemy).

Technologies: ES6, Express, Handlebars.

## Install

````bash
npm install
````

## Usage

Start web server:
````bash
npm run start
````
